Set Home Action

Video Tutorial

The primary purpose of this module is to allow any user to set their homepage with the help of a click on the Home button, so that whenever they log back in again, they start from the page that was set earlier.

Prerequisites before Installation

There are no prerequisites required before the Installation

Configurations

Easy configurantion Just install the module and a Home button will be added on the top right section of the screen along with the username as shown below
