Something More in Payslip!

Overview

This will help you in Odoo payslip generation. It adds some extra features to the existing payslip such as showing gross & net salary in list view and confirm button etc.

Displays Gross and net Salary in the list view. This will help you to analyse multiple payslips in single view. Net and gross salary are considered from the category code of payslip lines.

  • Gross salary is considered if category code is GROSS
  • Net salary is considered if category code is NET
Also added button near the gross and net salary to confirm multiple payslips easily.

Displays image of employee and the user who created the payslip.

Displays Gross and net Salary inside the batch. Then confirm button to confirm easily.

In Odoo, we need to update Gross and Net salary manually if any payslip line amount is modified.
So this module added a new button below payslip lines to update Gross and Net salary.
Below are calculations ...

  • Gross Salary Calculation ⇨ BASIC + ALW (Allowance)
  • Net Salary Calculation ⇨ BASIC + ALW (Allowance) - DED (Deduction)
