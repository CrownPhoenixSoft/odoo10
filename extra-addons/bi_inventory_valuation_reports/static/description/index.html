Stock Valuation Report

This module will Generate PDF Report and XLS File for Inventory valuation(Product Category).

Key Features:
  • Easily generate PDF Report And XLS file for stock valuation inventory.
  • Easily generate PDF Report in detail with and without product.
  • Easily generate XLS file for stock valuation with summarized data.

Menu

To get Report "Inventory Valuation Report" go to Inventory->Reports->Inventory Valuation Report.When you click on this you will get the Wizard.


Wizard

When you click to the Menu you will get this Wizard,in this Wizard you have to select particular duration and warehouse for Generating Report of inventory valuation.


After selecting Print PDF Option in Wizard you will get this PDF Report.



After selecting Print Excel Option you will get this Wizard.


After selecting Print Excel Option in Wizard you will get this XLS Report.
